MCPinger Ping any Minecraft server.
Invite Vote Support server Donate

MCPinger is a Discord bot, used to view the status of any Minecraft server. It shows the MOTD, player count, version, software and icon.

You may also view any Minecraft user's skin or UUID, create custom advancements and track the status of a server. Customize settings to visualise the ping embed to your likings.

Commands
✕

mc.ip <hostname>

Set the default Minecraft server. (Admin-only)

Arguments (optional)

--bedrock

Example

mc.ip hypixel.net mc.ip play.nethergames.org --bedrock

mc.ping

Ping the default Minecraft server.

Aliases

mc.p | mc.status | mc.bedrock | mc.b

mc.ping <hostname>

Ping any Minecraft server by its IP address.

Aliases

mc.p | mc.status | mc.bedrock | mc.b

Arguments (optional)

--bedrock

Example

mc.ping hypixel.net mc.ping play.nethergames.org --bedrock mc.bedrock play.nethergames.org

mc.multiping <hostname> <hostname>

Ping multiple Minecraft servers by their IP addresses.
Check the status of up to 6 servers simultaneously. (Not compatible with bedrock edition servers.)

Example

mc.multiping hypixel.net 2b2t.org

mc.skin <username>

View any Minecraft user's skin.

Aliases

mc.s

Example

mc.skin WouterK12

mc.uuid <username>

View any Minecraft user's UUID.

Example

mc.uuid WouterK12

mc.advancement <advancement>

Create a custom advancement / achievement.

Aliases

mc.a | mc.achievement

Example

mc.advancement mc.advancement Consumed 10 Pizzas! mc.advancement --20 You mined stone!

mc.tracker <log/embed/voice> <hostname>

Track the status of any Minecraft server. (Admin-only)
A check is performed every 15 minutes.

Types

Log the status, view it in an auto-updating embed or in the name of a voice channel.

The tracker will disable after 200 failed checks, when the server is offline. (That's 2 days and 2 hours)

Aliases

mc.t | mc.track

Arguments (optional)

--bedrock

Example

mc.tracker mc.tracker help mc.tracker log hypixel.net mc.tracker embed play.nethergames.org --bedrock

mc.settings <enable/disable> <field>

Customize what to show on a ping embed. (Admin-only)

Aliases

mc.set | mc.setting | mc.field | mc.fields

Fields

motd | players | players_list | version | software | server_icon | srv_host | mask_host | all

Example

mc.settings mc.settings disable software

mc.prefix <prefix>

Set the prefix. (Admin-only)

Example

mc.prefix !

Tip

To see what prefix you currently have, tag @MCPinger

mc.statistics

View MCPinger's statistics.

Aliases

mc.stats

mc.clear

Clear your data. (Admin-only)

mc.help

View a list of commands.
