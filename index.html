MCPinger

MCPinger is used to view the status of any Minecraft server. It shows the MOTD, player count, version and software.

You may also view any Minecraft user's skin or UUID, create custom advancements and track the status of a server.

Take a look at the commands below:

Commands

mc.ip <hostname>

Set the default Minecraft server. (Admin-only)

Aliases

mc.a | mc.be | mc.test

Example

mc.ip hypixel.net

mc.ping

Ping the default Minecraft server.

mc.ping <hostname>

Ping any Minecraft server by it's IP address.
