MCPinger Ping any Minecraft server.

MCPinger is a Discord bot, used to view the status of any Minecraft server. It shows the MOTD, player count, version, software and icon.

You may also view any Minecraft user's skin or UUID, create custom advancements and track the status of a server. Customize settings to visualise the ping embed to your likings.

Commands

mc.ip <hostname>

Set the default Minecraft server. (Admin-only)

Example

mc.ip hypixel.net

mc.ping

Ping the default Minecraft server.

Aliases

mc.p

mc.ping <hostname>

Ping any Minecraft server by it's IP address.

Aliases

mc.p

Example

mc.ping hypixel.net

mc.skin <username>

View any Minecraft user's skin.

Aliases

mc.s

Example

mc.skin WouterK12

mc.uuid <username>

View any Minecraft user's UUID.

Example

mc.uuid WouterK12

mc.advancement <advancement>

Create a custom advancement / achievement.

Aliases

mc.a | mc.achievement

Example

mc.advancement Consumed 10 Pizzas!

mc.tracker <enable/disable> <hostname>

Track the status of any Minecraft server. (Admin-only)
A check is performed every 30 minutes. If the status has changed, a notification is send in the channel in which the tracker was enabled. The tracker will disable after 100 failed checks, when the server is offline. (That's 2 days and 2 hours)

Aliases

mc.t | mc.track

Example

mc.tracker mc.tracker enable hypixel.net

mc.settings <enable/disable> <field>

Customize what to show on a ping embed. (Admin-only)

Aliases

mc.set | mc.setting | mc.field | mc.fields

Example

mc.settings mc.settings disable software

mc.suggest <suggestion>

Suggest a new feature.

Example

mc.suggest You should add a command that does ...
